Voici les instructions.
Il faudra faire un commit GIT pour chaque nouvelle fonctionnalité ajoutée à l'application.

  • Mettre en place in petit script python3 permettant d'avoir un serveur web et une vue REST qui retourne des données JSON simple de ce type:
    [{'name': 'sylvain', 'age': 29, 'sex': 'h'}, {'name': 'michele', 'age': 20 'sex': 'f'}]
    Tu peux ajouter plus de données.
  • Mettre en place une vue angularjs qui affiche un tableau avec ces données: /#!/users
  • Afficher l'age en vert si plus de 25 ans / rouge sinon
  • Trier les données du tableau en JS (via un filter) sur la colonne age
  • Mettre un champ input de type texte permettant de rechercher des prénoms. Le contenu du tableau va afficher uniquement les prénoms qui correspondent. La recherche devra être une recherche python.
  • Mettre en place une vue angularjs qui affiche le détail d'un utilisateur /#!/users/{id}
  • Mettre en place de la CSS dans le fichier SCSS pour que la page soit jolie (menus / contenu / tableaux ....)
  • Ajouter un style différent sur les liens "Accueil" / "Voir la todo list" quand les onglets sont actifs
  • Mettre à jour la documentation dans README.rst
